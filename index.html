Instructions

First, look up the demo, which shows the finished procuct. This memory cards count contains 24 cards, and the cards'back have numbers as images. The purpose of the gave is to turn two cards back over if a match is not made. If a match is made, then both cards in the match remained flipped. When all the matches are found, the program reshuffles the entire board, so all the cards are in new locations.
